Added unordered list to html

  • Untracked: Bestand is nog niet gevolgd door Git
  • Tracked: Bestand is gevolgd door Git maar er zijn geen wijzigingen
  • Modified: Bestand is gewijzigd ten opzichte van de vorige commit
  • Staged: Gewijzigd bestand is toegevoegd aan de staged changes voor de volgende commit
  • Committed: Gewijzigd bestand is gecommit naar de repository
